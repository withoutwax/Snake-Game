# Snake-Game
The is a simple Snake Game.

Created with Java language in Processing.
You can navigate easily and also change theme accordingly

Keyboard - 
Controls: UP, DOWN, LEFT , RIGHT
P - Pause
O - Resume
R - Restart

1 - Theme01
2 - Theme02
3 - Theme03
4 - Theme04
5 - Theme05 
Use arrow keys on the keyboard as controls
